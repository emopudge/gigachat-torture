Задача: Бинарная классификация. Оформит ли клиент банковский депозит по итогам маркетинговой кампании.

data - директория с данными для решения задачи предсказания оформления банковского депозита, содержит:

a. train.csv - обучающая выборка:
    client_id - уникальный идентификатор клиента
    target - целевая переменная; принимает значение 1, если клиент оформит банковский депозит по итогам маркетинговой кампании, и 0 в противном случае

b. test.csv - тестовая выборка:
    client_id - уникальный идентификатор клиента
    target - целевая переменная; принимает значение 1, если клиент оформит банковский депозит по итогам маркетинговой кампании, и 0 в противном случае

c. client_data.csv - таблица с признаками клиентов:
    client_id - уникальный идентификатор клиента
    age - возраст клиента
    job - тип занятости ("admin.","blue-collar","entrepreneur","housemaid","management","retired","self-employed","services","student","technician","unemployed","unknown")
    marital - семейное положение ("divorced","married","single","unknown")
    education - уровень образования ("basic.4y","basic.6y","basic.9y","high.school","illiterate","professional.course","university.degree","unknown")
    default - наличие просрочки по кредиту ("no","yes","unknown")
    housing - наличие жилищного кредита ("no","yes","unknown")
    loan - наличие потребительского кредита ("no","yes","unknown")
    contact - тип коммуникации при последнем контакте в рамках текущей маркетинговой кампании ("cellular","telephone")
    month - месяц последнего контакта
    day_of_week - день недели последнего контакта
    campaign - количество контактов с клиентом в рамках текущей маркетинговой кампании, включая последний
    pdays - количество дней с момента предыдущего контакта в прошлой маркетинговой кампании; значение 999 означает отсутствие предыдущих контактов
    previous - количество контактов с клиентом до текущей маркетинговой кампании
    poutcome - результат предыдущей маркетинговой кампании ("failure","nonexistent","success")
    emp.var.rate - коэффициент изменения занятости, квартальный макроэкономический показатель
    cons.price.idx - индекс потребительских цен, месячный макроэкономический показатель
    cons.conf.idx - индекс потребительского доверия, месячный макроэкономический показатель
    euribor3m - ставка Euribor на 3 месяца, дневной макроэкономический показатель
    nr.employed - численность занятых, квартальный макроэкономический показатель
